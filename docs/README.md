## AutoGrader 사용 도움말

AutoGrader는 학생 또는 교사가 직접 문제집을 등록하고, 모바일에서 빠르게 답안을 채점할 수 있도록 만든 웹 앱입니다.

### 1. 로그인

- Google 계정으로 로그인합니다.
- **중동고등학교 계정(@joongdong.hs.kr)**만 허용됩니다. 허용되지 않은 계정으로 로그인하면 접근이 차단됩니다.

### 2. 문제집 목록 확인

- 로그인 후 대시보드에서 저장된 문제집 목록을 확인할 수 있습니다.
- 문제집 카드에는 제목, 문항 수, 저장/수정 시각이 표시됩니다.

### 3. 문제집 만들기 (PC 권장)


> [!IMPORTANT]
> 문제집 하나에는 최대 100개의 문항을 등록할 수 있습니다. 100문항이 넘는 문제집을 등록하는 경우 분할하여 등록해야 합니다.


1. 대시보드에서 `새 문제집 등록` 버튼을 누릅니다.
2. 제목과 문제 개수를 입력합니다.
3. 문제 행에서 각 문항의 유형과 정답을 입력합니다.
4. `문제집 저장` 버튼을 누르면 저장됩니다.

> [!TIP]
> PC 환경에선 객관식의 경우엔 1-5를, 단답형의 경우엔 엔터를 누르면 다음 행으로 넘어가 입력이 편리합니다.

| 유형        | 설명                                                   | 입력 방식   |
| ----------- | ------------------------------------------------------ | ----------- |
| `객관식`    | 정답은 1~5 중 하나만 입력할 수 있습니다.               | 숫자 1~5    |
| `단답형`    | 짧은 텍스트 정답을 입력합니다.                         | 텍스트 입력 |
| `채점 불가` | 수동 확인이 필요한 문항으로, 정답을 입력하지 않습니다. | 입력 없음   |

### 4. 문제집 수정 (PC 권장)

- 대시보드에서 문제집 카드의 `수정` 버튼을 누르면 해당 문제집을 불러와 수정할 수 있습니다.

### 5. 채점 시작

1. 문제집 카드에서 `채점 시작`을 누릅니다.
2. 채점 시작 문제 번호를 입력합니다.
3. `채점 시작`을 누르면 해당 번호부터 순서대로 채점이 시작됩니다. (100문항 중 11번부터 채점하려면 시작 번호에 `11`을 입력합니다)

### 6. 채점 중 사용법

- `1~5` 객관식 버튼을 누르면 해당 답으로 기록됩니다.
- `단답형`은 답을 입력한 뒤 `확인`을 누르거나 Enter를 누릅니다.
- `스킵` 버튼은 이번 문항을 건너뜁니다.
- `이전` 버튼으로 이전 문항으로 돌아갈 수 있습니다.
- `채점 완료` 버튼을 누르면 현재 위치에서 채점을 종료합니다.

### 7. 채점 결과 보기

- 채점이 끝나면 결과 화면에서 정답/오답/스킵/채점 불가 개수를 확인할 수 있습니다.
- `리스트 보기`를 누르면 각 문항의 답안과 정답을 상세하게 확인할 수 있습니다.
- `목록으로`로 돌아가거나 `다시 채점`으로 다시 시작할 수 있습니다.
